# EmployeeManagement
This is a Employee management system.
There are five example employee information for presenting.

## Create:
User can create a new enployee profile with photo.

## View(Read):
Table display all employee basic information.
User can click employee in the table and click "view". It will display more detailed information including employee photo.

## Search:
User can choose the specific filed or all fields(whole table) in combobox to decide the range of search

## Delete:
User can select one row to delete employee

## Edit(Update):
With view spcific information function, user can edit employee information. Also, user can update photo by upload new photo.
